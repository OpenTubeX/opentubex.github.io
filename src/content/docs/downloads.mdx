OpenTubeX is available on Windows 10+, macOS 12+, and Linux.

## Official releases

- [GitHub Releases](https://github.com/OpenTubeX/OpenTubeX/releases)
- [AUR (opentubex-bin)](https://aur.archlinux.org/packages/opentubex-bin/)

## Automated builds

Use these pre-release builds at your own risk.

- [GitHub Actions builds](https://github.com/OpenTubeX/OpenTubeX/actions?query=workflow%3ABuild)
- [AUR (opentubex-git)](https://aur.archlinux.org/packages/opentubex-git/)

> You need a GitHub account to download builds from GitHub Actions.
> If you don't have one, you can use [nightly.link](https://nightly.link/OpenTubeX/OpenTubeX/workflows/build/development).
